== Description ==

<h3>Customize Dokan Dashboard Endpoints</h3>

Customize your default Customize Dokan Dashboard Endpoints. Reorder them , hide existing core endpoints. 


<p>Want to add custom endpoints with shortcode ? check <a href="https://www.sysbasics.com/go/dokanendpoints/" >Customize Dokan Dashboard Endpoints Pro
</a></p>

<h4> Customize Dokan Dashboard Endpoints features </h4>
- Modify existing endpoints.
- Add custom links to your dokan vendor dashboard.
- Drag and drop UI.

<h4> Customize Dokan Dashboard Endpoints free version features</h4>
- Add new Link or Menu to exitings dokan dashboard endpoints
- Reorder core  dokan dashboard endpoints
- Hide Core Dashboard Endpoints
- Drag and Drop UI
- Add Icon to existing dokan dashbaord Endpoints
- Adding Extra Menu on Vendor Dashboard
