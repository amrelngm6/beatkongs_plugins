== Installation ==

Extract the zip file and just drop the contents in the wp-content/plugins/ directory of your WordPress installation and then activate the Plugin from Plugins page.

== Frequently Asked Questions ==

= Q. Can I get new design of vendor templates? =
A. Yes, you will get new vendor store templates.

= Q. Does this plugin integrate with Other multi vendor plugin? =
A. Not for the moment. We have plans to integrate with other multi vendor plugin via an add-on in the near future.

= Q. Is Vendor templates Dokan responsive? =
A. Yes, Vendor templates for Dokan plugin is fully responsive.
